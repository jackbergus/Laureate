## Keys

|  Keys   | Useage  |
|  ----  | ----  |
| F1  | Reset |
| F2  | Init camera |
| F3  | Print event queue |
| F4  | Toggle debug state machine |
| W,A,S,D,shift,space  | Camera move |
| up,down,left,right  | player move |
| z  | Attack |
| x  | Stop |
| c  | Toggle the pet strategy mode |
